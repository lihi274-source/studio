# **App Name**: Viajes Explorador

## Core Features:

- Pestaña de Vuelos: Permite a los usuarios buscar y comparar vuelos de diferentes aerolíneas.
- Pestaña de Hoteles: Permite a los usuarios buscar y reservar hoteles según ubicación, precio y servicios.
- Pestaña de Excursiones: Ofrece una variedad de excursiones y actividades en diferentes destinos.
- Generador de Itinerarios: Un asistente de IA para planificar viajes, considerando las preferencias del usuario y ofreciendo sugerencias personalizadas de vuelos, hoteles y actividades. The LLM will act as a tool, and it will optionally incorporate facts depending on their alignment with a user's request.
- Soporte Multi-Idioma: La aplicación está completamente traducida al español.

## Style Guidelines:

- Color primario: Azul suave (#64B5F6) para reflejar confianza y tranquilidad.
- Color de fondo: Beige claro (#F5F5DC) para una sensación acogedora.
- Color de acento: Naranja (#FFB74D) para resaltar elementos importantes y llamadas a la acción.
- Fuente para títulos: 'Playfair', una fuente serifada elegante.
- Fuente para cuerpo de texto: 'PT Sans', una fuente sans-serif para asegurar la legibilidad.
- Utilizar iconos claros y reconocibles para las diferentes secciones y funciones.
- Diseño limpio y moderno, con un enfoque en la facilidad de navegación y la accesibilidad.
- Animaciones sutiles para mejorar la experiencia del usuario, como transiciones suaves entre páginas y efectos hover en los elementos interactivos.
